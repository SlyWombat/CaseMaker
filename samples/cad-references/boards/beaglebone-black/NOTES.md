# BeagleBone Black — CAD references

- `BBB_SCH.pdf` — full schematic (Rev C).
- `BBB_SRM.pdf` — System Reference Manual including mechanical drawings.
- `BBB_RevC_ASSY.pdf` — assembly drawing (mechanical view).
- `upstream-README.md` — upstream README from beagleboard/beaglebone-black.
