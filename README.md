# Case Maker

A web-based Single Page Application for designing custom 3D-printable enclosures for microcontrollers and single-board computers (ESP32, Arduino, Raspberry Pi, and custom boards).

See [docs/casemaker.md](docs/casemaker.md) for the full design brief.

## Status

Pre-implementation. Design and architecture phase.

## Stack (planned)

- React + Vite (SPA)
- Three.js (3D viewport)
- Manifold WASM (CSG geometry)
- Zustand (state)
- Playwright (end-to-end tests)
